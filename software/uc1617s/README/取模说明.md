## PCtoLCD2002 取模设置

### 列式（用于无后缀函数）

```
┌─────────────────────────────────┐
│  PCtoLCD2002 设置               │
│                                 │
│  取模方式: ● 逐列式             │
│  取模走向: ● 逆向 (低位在前)    │
│  输出格式: ● C51                │
│  宽度: 16  高度: 16             │
└─────────────────────────────────┘
```



生成结果（"你"）：

```
/* 逐列式, 逆向 (LSB=顶), 每列1字节, 共32字节 */
/* page0 (行0~7), 每列1字节 */
{0x00,0x00,0x00,0x00,0x00,0x02,0x00,0x06,
 0x00,0x0C,0x38,0x18,0x38,0x30,0x38,0x60,
/* page1 (行8~15), 每列1字节 */
 0x38,0xC0,0x39,0x80,0x3F,0x00,0x3E,0x00,
 0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,},

第0列: 0x00 (全黑)
第5列: 0x02=0000_0010 → bit1=1 → 行1=亮
              ↑bit0=行0(顶)
```



对应代码：

```
/* 列式取位 */
bits & (1 << bit)   /* bit0=顶 */
```

### 设置速查表



```
函数类型          取模方式   取模走向     取位公式
─────────────────────────────────────────────────────
行式 (_ex)        逐行式    顺向(高位)   data & (0x80 >> col)
列式 (无后缀)     逐列式    逆向(低位)   data & (1 << bit)
```



### 验证方法



画一个简单字符（如 "十"）对比：



```
行式输出 (顺向, MSB=左):
  第0行: 0x08,0x00  → 0000_1000 → 第4列=1
  第7行: 0xFF,0x80  → 1111_1111 → 全亮

列式输出 (逆向, LSB=顶):
  第7列: 0xFF       → 1111_1111 → 行0~7全亮 (中间竖线)
  第0列: 0x00       → 全黑
```



在 LCD 上显示正确就说明设置对了。
